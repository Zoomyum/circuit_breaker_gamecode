This game is inspired by the theme Overload
The game consists of 6 different nodes which has ever increasing pressure, which the goal of the game is to remove the pressure.
Each node has different rate of change in pressure and only 1 node can remove the pressure for good, which after that requires a cooldown.
The goal is to survive the longest
